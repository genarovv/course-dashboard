## Решения CEO по развилкам митинга (2026-07-09)

1. **Модель ядра:** `deepseek-v4-flash` (≈$1.3–1.5/мес; pro — fallback, если пилот Фазы 0 покажет пропуски).
2. **Append-only:** BEFORE UPDATE/DELETE-триггеры только на `artifact_snapshot` + `coherence_verdict` (доказательное ядро FR-9); остальное — дисциплина store.py.
3. **Токены:** env-var / файл chmod 600; колонка `encrypted_token` и криптомеханизм из БД удаляются; NFR-3 переформулировать с записью «почему не строим крипто».
4. **Утверждены все 9 вычёркиваний** из ARCHITECTURE.md (список — протокол митинга, §«Что вычеркнуть»); каждое — с записью «почему не строим X».
5. **FR-11/StepQualityCard:** файл модели в v1-коде не создавать; сущность остаётся в data-model как будущее v2.
