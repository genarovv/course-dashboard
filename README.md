# Course Dashboard

Демо-проект курса «Управление агентной разработкой» ([шаблон контекста для студентов — отдельный репозиторий](https://github.com/genarovv/ai-agents-project-context)).

**Продукт:** дашборд преподавателя, который сам читает `.md`-артефакты из публичных репозиториев студентов, строит матрицу «кто что сдал» и LLM-агентом подсвечивает **разрывы связности** между артефактами (требования ↔ схема данных ↔ архитектура ↔ …). Преподаватель ничего не вводит руками и смотрит только в подсвеченные точки.

Проект ведётся агентной разработкой live на занятиях курса и одновременно служит эталоном той самой структуры артефактов, которую он проверяет у студентов.

## Статус

Требования готовы (PRD v2.2). Кода нет: впереди схема данных (занятие 6), архитектура (занятие 7), затем разработка. Ядро — за гейтом живой валидации (Фаза 0).

> Все интервью в основе требований — синтетические (ИИ играл персону); требования — приоритизированные гипотезы. Это учебная честность курса, зафиксированная в PRD §0.
