# Golden set — эталонные пары для проверки ядра FR-5

> Механизм тестируемости LLM-ядра (ADR-001, п.6): при смене рубрики или модели прогоняются все пары;
> расхождение с эталонным вердиктом = регрессия. Метрика доверия — ≤20% ложных разрывов (PRD §4).
> Пары добавляются по мере курирования: каждый подтверждённый/опровергнутый преподавателем разрыв — кандидат сюда.

## Эталонный промпт-каркас рубрики (v1, 2026-07-02)

Роль: агент-проверяющий связности, работает строго по правилу. Правило ребра: «B обязан опираться на A: значимая сущность A либо отражена в B (синонимы засчитываются и называются), либо исключена явно (out-of-scope с обоснованием), либо потеряна. Разрыв = ≥1 потерянная значимая сущность». Выход: ВЕРДИКТ / УВЕРЕННОСТЬ / СЧЁТЧИКИ / ПОДСВЕЧЕННЫЕ ТОЧКИ (≤5: сущность | цитата A ≤15 слов | что искал в B и почему не засчитал) / ЗАМЕТКИ ≤3 строк.

## Пары

### GS-1 (П1): user stories преподавателя → PRD — эталон: НЕТ РАЗРЫВА
- A: `product/user-stories/01-user-stories-преподаватель.md` · B: `product/prd.md` (v2 от 2026-07-02)
- Прогон 2026-07-02: вердикт НЕТ РАЗРЫВА, уверенность высокая, 25 сущностей: 21 отражено, 4 исключено явно, 0 потеряно. ✅ совпал с эталоном.
- Ценная спорная точка (то, что должен уметь находить хороший прогон): дефиниционный дрейф «слепой зоны» — US-B3 «нет новых артефактов N занятий» vs FR-6 «репозиторий не добавлен/недоступен».
- **Обновление 2026-07-02:** рассинхрон починен — US-B3 переопределена как «вне обзора» (невидимость), сигнал тишины передан US-B4/FR-7. При следующем прогоне GS-1 эта точка воспроизводиться не должна; её появление = ложная сработка.

### GS-2 (П3): персона Оксана (другой преподаватель) → user stories — эталон: РАЗРЫВ
- A: `product/personas/04-преподаватель-оксана.md` · B: `product/user-stories/01-user-stories-преподаватель.md`
- Прогон 2026-07-02: вердикт РАЗРЫВ, уверенность высокая, 9 сущностей: 1 отражена, 0 исключено, 8 потеряно. ✅ совпал с эталоном.
- Ключевая точка-эталон: «Оксана потеряна молча — в отличие от Дамира/Алии/Рината/Елены, явно исключённых в out-of-scope». Прогон, не находящий эту точку, — деградация.

## Что показал лексический бейзлайн (для истории)
R1 (идентификаторы): 100% на артефактах с ID-конвенциями, вырождается на персонах/интервью. R2 (backticks/bold/заголовки + стемминг): порог «≥1 сущность» пропускает GS-2; порог «покрытие ≥30%» ловит, но точки бессмысленны (рубрики карточки, не сущности). Вывод — ADR-001.
