## Ключевые решения

1. Дашборд не персонифицирует студентов — только артефакты их проектов
2. ~~Дашборд публичный псевдонимизированный + приватный именной вид преподавателя~~ → **SUPERSEDED (2026-07-02):** v1 — только приватный вид преподавателя, публичного вида нет вообще; **межстуденческого сравнения нет by design** (BR-4)
3. ~~Сроки сдачи — главная проблема, связность — производная~~ → **СНЯТО.** В синтетическом интервью сроки прощаются, ядро боли — **разрыв связности** (FR-5a). Прежняя формулировка противоречила интервью и тащила дрейф к симптому. *(Гипотеза до Фазы 0 — живого custdev.)*
4. Инструмент обязан читать данные из репозиториев автоматически (без ручного ввода данных; курирование находок — FR-10 — допустимо, BR-1)
5. Механика (наличие артефактов + структурный разрыв) автоматизируется, оценка смысла остаётся преподавателю
